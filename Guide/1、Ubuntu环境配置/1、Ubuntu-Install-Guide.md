### 1.1 设置root密码并切换至root 【可选】
```bash
sudo passwd root && su
```
### 1.2 配置时区
```bash
sudo timedatectl set-timezone Asia/Shanghai
```
### 1.3 更新软件
```bash
sudo apt update -y && sudo apt upgrade -y 
```
