## 2、配置全局 dl 目录

### 2.1 修改 defconfig 目录
```bash
sudo mkdir -p /opt/buildroot/download
```

### 2.1 创建目录
```bash
sudo mkdir -p /opt/buildroot/download
```

### 2.2 设置所属组为 users【也可设置为对应的开发组】
```bash
sudo chown -R :users /opt/buildroot/download
```

### 2.3 设置 download 目录权限为全局可读写
```bash
sudo chmod 2775 /opt/buildroot/download
```
