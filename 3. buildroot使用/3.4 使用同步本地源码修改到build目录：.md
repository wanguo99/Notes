## 1. 使用本地源码的两种方式
### 1. 在package/xxx目录下创建.local文件
1. 创建文件package/es-middleware/es-middleware.mk.local
2. 写入以下内容
```Makefile
# ES-Middleware local override
# This file overrides the default package settings to use local source code

ES_MIDDLEWARE_OVERRIDE_SRCDIR = $(TOPDIR)/../ES-Middleware
```

### 2. 在TOPDIR目录下创建local.mk文件
1. 在buildroot根目录创建文件local.mk
2. 写入以下内容
```Makefile
# ES-Middleware local override
# This file overrides the default package settings to use local source code

ES_MIDDLEWARE_OVERRIDE_SRCDIR = $(TOPDIR)/../ES-Middleware
```


同步本地源码修改到build目录：
```bash
  make <pkg>-reconfigure      # - Restart the build from the configure step
```
