# 3.4 使用同步本地源码修改到 build 目录：

## 1. 使用本地源码的两种方式

### 1. 在 package/xxx 目录下创建.local 文件
1. 创建文件 package/es-middleware/es-middleware.mk.local
2. 写入以下内容

```Makefile
# ES-Middleware local override
# This file overrides the default package settings to use local source code

ES_MIDDLEWARE_OVERRIDE_SRCDIR = $(TOPDIR)/../ES-Middleware
```

### 2. 在 TOPDIR 目录下创建 local.mk 文件
1. 在 buildroot 根目录创建文件 local.mk
2. 写入以下内容

```Makefile
# ES-Middleware local override
# This file overrides the default package settings to use local source code

ES_MIDDLEWARE_OVERRIDE_SRCDIR = $(TOPDIR)/../ES-Middleware
```

同步本地源码修改到 build 目录：

```bash
  make <pkg>-reconfigure      # - Restart the build from the configure step
```
