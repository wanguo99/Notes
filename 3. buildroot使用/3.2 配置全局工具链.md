# 3.2 配置全局工具链

## 1. 编译 SDK

### 1.1 修改 defconfig 中的 BR2_DL_DIR
```bash
cat >> ${HOME}/CSPD/buildroot-external/configs/h200_100p_am625_defconfig <<EOF
BR2_DL_DIR="/opt/buildroot/download"
EOF
```

### 1.2 编译 SDK
```bash
BR2_EXTERNAL=${HOME}/CSPD/buildroot-external make h200_100p_am625_defconfig
```

```bash
make sdk
```

```bash
sudo mkdir -p /opt/buildroot/download
```

### 1.3 安装到服务器
```bash
sudo mkdir -p /opt/buildroot/download
```
