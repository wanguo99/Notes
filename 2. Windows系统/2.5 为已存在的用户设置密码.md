# 2.5 为已存在的用户设置密码

在 Windows 10 中为已存在的用户设置密码，主要有以下几种常用且便捷的方法：

### 方法一：通过“设置”应用（最常用）

1. 按下键盘上的 `Win + I` 组合键打开“设置”窗口。
2. 点击“账户”选项，然后在左侧导航栏中选择“登录选项”。
3. 在右侧找到“密码”区域，点击“添加”按钮。
4. 根据系统提示，依次输入您希望设置的“新密码”、“确认新密码”，并可选填一句有助于记忆的“密码提示”。
5. 点击“下一步”，然后点击“完成”即可生效。

### 方法二：使用快捷键（最快捷）

1. 确保电脑处于已登录桌面的状态，同时按下键盘上的 `Ctrl + Alt + Delete` 键，调出安全选项界面。
2. 在弹出的菜单中选择“更改密码”。
3. 在旧密码栏留空（因为当前未设密码），然后输入新密码并确认。
4. 按回车或点击确认箭头，即可完成密码的设置。

### 方法三：通过控制面板（传统方式）

1. 在任务栏搜索框中输入“控制面板”并打开。
2. 将右上角的查看方式设为“类别”，然后点击“用户账户” -> “管理其他账户”。
3. 在账户列表中，点击您需要设置密码的那个用户名。
4. 选择“创建密码”或“更改密码”，输入新密码及密码提示后，点击“更改密码”按钮保存即可。

### 💡 安全建议

为了保障电脑的数据安全，建议您设置一个强密码。通常要求密码长度至少为 8 位，并且包含大写字母、小写字母、数字和特殊符号中的至少三类组合。同时，请确保密码提示语不包含真实的个人信息，以免被他人轻易猜出。
