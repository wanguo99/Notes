以下是 Git 中英文界面切换的完整设置指南，按推荐程度排序：

### 1. Git 配置法（推荐，Git ≥ 2.39）【验证无效】

通过 `core.language` 配置项直接控制，仅影响 Git，不改变系统语言。

| 操作        | 命令                                          |
| :-------- | :------------------------------------------ |
| **设为英文**  | `git config --global core.language en`      |
| **设为中文**  | `git config --global core.language zh_CN`   |
| **仅当前仓库** | 将 `--global` 替换为 `--local`                  |
| **取消设置**  | `git config --global --unset core.language` |

> ⚠️ 使用 `git --version` 确认版本 ≥ 2.39，否则此配置无效。

### 2. 环境变量法（通用，所有版本适用）【验证有效】

当 Git 版本较旧或 `core.language` 不生效时使用。优先级：`LANGUAGE` > `LC_ALL` > `LANG`。

#### Linux / macOS

```bash
# 设为英文
export LANGUAGE=en_US.UTF-8
export LC_ALL=en_US.UTF-8

# 设为中文
export LANGUAGE=zh_CN.UTF-8
export LC_ALL=zh_CN.UTF-8

# 永久生效：追加到 ~/.bashrc 或 ~/.zshrc 后执行 source
```

#### Windows PowerShell

```powershell
# 设为英文
[System.Environment]::SetEnvironmentVariable("LANGUAGE", "en_US", "User")
[System.Environment]::SetEnvironmentVariable("LC_ALL", "en_US", "User")

# 设为中文
[System.Environment]::SetEnvironmentVariable("LANGUAGE", "zh_CN", "User")
[System.Environment]::SetEnvironmentVariable("LC_ALL", "zh_CN", "User")

# 修改后需重启终端/IDE 生效
```

#### Windows CMD

```cmd
:: 设为英文
setx LANGUAGE en_US
setx LC_ALL en_US

:: 设为中文
setx LANGUAGE zh_CN
setx LC_ALL zh_CN
```

### 3. Git for Windows 安装器法（仅限 Windows）

如果安装时选错了语言，最彻底的方式是重新运行安装程序：

- **改英文**：重新安装 → 组件选择中取消 "Chinese Language Pack"
- **改中文**：重新安装 → 勾选 "Chinese Language Pack"

### 4. 临时单次命令

不改任何配置，仅让某条命令以指定语言输出：

```bash
# 单次英文
LANGUAGE=en git status

# 单次中文
LANGUAGE=zh_CN git log
```

### ✅ 验证是否生效

```bash
git status
# 英文输出: On branch main, nothing to commit
# 中文输出: 位于分支 main，无文件要提交
```

### 📋 快速对照表

|场景|英文|中文|
|:--|:--|:--|
|Git ≥ 2.39 全局设置|`git config --global core.language en`|`git config --global core.language zh_CN`|
|旧版 Git / 通用|`export LANGUAGE=en_US.UTF-8`|`export LANGUAGE=zh_CN.UTF-8`|
|Windows 永久设置|`setx LANGUAGE en_US`|`setx LANGUAGE zh_CN`|
|临时单次|`LANGUAGE=en git <cmd>`|`LANGUAGE=zh_CN git <cmd>`|
|Windows 安装包|重装取消中文包|重装勾选中文包|

> 💡 **提示**：中文翻译覆盖率并非 100%，部分命令即使设置了中文仍可能显示英文，这属于正常现象。
