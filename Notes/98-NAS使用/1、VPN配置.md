**群晖自己搭建 VPN 的最简单完整教程（2026最新推荐）**

群晖自带 **VPN Server** 套件，操作非常简单，主要支持 **OpenVPN**（推荐）、L2TP/IPSec。**WireGuard** 需要额外 Docker 方式实现。

---

### **推荐方案优先级**
1. **OpenVPN**（最稳，官方支持好，适合大多数人）
2. **WireGuard**（速度最快，推荐新手用 Docker + WG-Easy）
3. Tailscale / ZeroTier（零配置，但不是传统 VPN）

---

### **方法一：官方 VPN Server 搭建 OpenVPN（最推荐新手）**

1. **安装套件**
   - 登录群晖 DSM → **套件中心** → 搜索 **VPN Server** → 安装。

2. **配置 OpenVPN 服务**
   - 打开 **VPN Server**。
   - 左侧选择 **OpenVPN** → 勾选 **启用 OpenVPN 服务器**。
   - 设置参数：
     - **动态 IP 地址**：建议 `10.8.0.0/24`（不要和家里局域网冲突）。
     - **端口**：默认 `1194`（UDP）。
     - **最大连接数**：按需设置（推荐 5-10）。
     - **允许远程访问的网络**：勾选你想通过 VPN 访问的局域网网段（如 `192.168.1.0/24`）。
   - 点击 **应用**。

3. **添加 VPN 账号**
   - 切换到 **权限** 标签 → 为需要远程访问的用户勾选权限（建议新建专用 VPN 用户）。

4. **导出配置文件**
   - 在 OpenVPN 页面点击 **导出配置**，下载 `.ovpn` 文件。

5. **路由器端口转发（关键！）**
   - 登录你的光猫/路由器 → 端口转发（Virtual Server）：
     - 外部端口：1194 → 内部 IP（群晖 IP） → 内部端口：1194 → 协议：UDP。

6. **客户端连接**
   - **Windows**：下载 OpenVPN 客户端（官网），导入 `.ovpn` 文件即可连接。
   - **手机**：下载 OpenVPN Connect App，导入配置文件。

连接成功后，Windows 就能直接用 `\\群晖IP` 或 `\\QuickConnectID` 映射网络驱动器了，和在家一样。

### **方法二：WireGuard（速度更快，推荐）**

官方没有原生 WireGuard，可用 **Docker + WG-Easy**（带网页管理界面）。

1. 安装 **Container Manager**（原 Docker）。
2. 在 Container Manager 中搜索并运行 **wg-easy** 镜像（推荐 ghcr.io/wg-easy/wg-easy）。
3. 基本配置：
   - 环境变量 `WG_HOST` 填你的 DDNS 或公网 IP。
   - 端口映射 51820/UDP。
   - 通过网页（NAS IP:51821）管理客户端，一键生成二维码或配置文件。
4. 路由器端口转发 51820 UDP 到群晖。

WireGuard 速度明显比 OpenVPN 快，尤其适合大文件读写。

---

### **额外重要设置**

- **DDNS**（强烈建议）：
  - 控制面板 → **外部访问** → **DDNS** → 添加 Synology DDNS 或 No-IP 等。

- **防火墙**：
  - 群晖控制面板 → **安全性** → **防火墙** → 允许对应 VPN 端口。

- **安全性建议**：
  - 使用强密码 + 双因素认证。
  - 不要用 PPTP（已过时，不安全）。
  - 定期更新 DSM 和套件。
