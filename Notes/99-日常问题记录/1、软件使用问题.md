1. Q-dir / Source Insight 无法看到网络驱动器
	- 如果是从安装目录启动，权限存在问题
	- 关闭当前进程之后从开始菜单或桌面启动即可

2. VMware Ubuntu22.04 修改虚拟机核心数、内存大小之后，重启虚拟机，`ifconfig -a` 可以看到网卡，显示为down状态，图形界面设置内找不到对应的网卡
	- 原因未知
	- 执行以下命令，开启网络可以恢复正常
```shell
sudo nmcli networking on
```

3. 虚拟机内配置nvim主题之后，xshell无法正常显示色彩
	- 开启 Xshell 的“使用本色” (关键步骤)
		这是 Xshell 特有的设置，也是最容易被忽略的。Xshell 默认会尝试模拟颜色，导致真彩色丢失。
	- 点击 Xshell 菜单栏的 “工具” (Tools) -> “选项” (Options)。
	- 在左侧选择 “高级” (Advanced)。
	- 在右侧找到 “使用本色” (Use original colors) 选项，并勾选它。
	- 点击确定，然后重启 Xshell 或重新连接会话。
	- 原理：
		勾选此项后，Xshell 会强制直接显示应用程序发送的 RGB 颜色值，而不是将其映射到 Xshell 自己的调色板中。

4. Ubuntu24.04配置共享文件夹
```shell
# 手动挂载
sudo mount -t fuse.vmhgfs-fuse .host:/ /mnt/hgfs -o allow_other
# 自动挂载
echo '.host:/ /mnt/hgfs fuse.vmhgfs-fuse allow_other,defaults 0 0' >> /etc/fstab
```


5. 安装claude
```
在 Ubuntu 系统中安装 Claude Code（Anthropic 官方推出的命令行 AI 工具）非常便捷，主要有两种官方推荐的方式。你可以根据自己的需求选择其中一种：

### 🚀 方式一：官方一键脚本安装（最推荐，简单省心）

这种方式完全不需要你手动安装 Node.js 等环境，官方脚本会自动帮你搞定一切，非常适合新手或希望环境隔离的用户。

直接在 Ubuntu 终端中复制并运行以下命令：

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

脚本执行完毕后，它会自动将 Claude 安装到你的 `~/.local/bin` 目录下。如果终端提示找不到命令，只需要运行下面这行命令，把该路径加入到环境变量中即可：

```bash
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc && source ~/.bashrc
```

### 💻 方式二：通过 npm 安装（适合前端/Node.js 开发者）

如果你本身就是开发者，或者系统中已经配置好了 Node.js 环境，可以通过 npm 来安装。

1. **确保 Node.js 版本达标**： Claude Code 要求 Node.js 版本在 18.0 或更高。你可以在终端输入 `node --version` 检查。如果没有安装或版本过低，可以通过以下命令安装最新的长期支持版（LTS）：
    
    ```bash
    curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
    sudo apt install -y nodejs
    ```
    
2. **全局安装 Claude Code**：
    
    ```bash
    npm install -g @anthropic-ai/claude-code
    ```
    
    _避坑提示_：如果安装时出现 `EACCES` 权限报错，**千万不要直接用 `sudo` 强行安装**。推荐的做法是配置一个用户级的全局目录：
    
    ```bash
    mkdir -p ~/.npm-global
    npm config set prefix '~/.npm-global'
    echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
    source ~/.bashrc
    ```
    
    配置好之后，再次运行上面的 `npm install` 命令即可顺利安装。
    

---

### 🔑 安装后的认证与使用

无论你使用哪种方式安装，完成后的使用步骤都是一样的：

1. **验证安装**： 在终端输入 `claude --version`，如果能正常输出版本号，说明安装成功。
2. **启动并登录**： 输入 `claude` 启动工具。首次运行时，终端会显示一个验证码，并自动打开你的浏览器跳转到 Anthropic 官网。登录你的账号（需要 Claude Pro 订阅或 API 账户）并完成授权，终端就会自动连接成功。
3. **开始使用**： 你可以通过 `cd` 命令进入任意一个项目文件夹，再次输入 `claude` 启动。它会询问你是否创建 `CLAUDE.md` 文件，建议选择“是”。这个文件相当于项目的“说明书”，你可以把项目的构建命令、代码规范等写进去，Claude 就能记住这些上下文，更好地辅助你写代码。

_温馨提示_：由于 Anthropic 的服务器在海外，如果在连接或安装过程中遇到网络问题，可能需要你自行配置好网络代理环境。
