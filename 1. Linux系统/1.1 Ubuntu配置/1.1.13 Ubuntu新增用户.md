在 Ubuntu 系统中新增用户，通常有两种常用的命令：`adduser` 和 `useradd`。对于大多数日常使用，推荐使用更友好、交互式的 `adduser` 命令。

### 方法一：使用 `adduser` 命令（推荐）

这是 Ubuntu 官方推荐的创建用户方式。它会自动引导你完成设置密码、创建家目录等一系列操作。

**1. 创建新用户**

```bash
sudo adduser username
```

_(将 `username` 替换为你想要的用户名)_

执行后，系统会进行交互式提示：

- 要求输入并确认新用户的密码（密码输入时不会显示字符）。
- 提示填写用户信息（如全名、房间号、电话等），如果不需要填写，直接按回车键跳过即可。
- 最后询问信息是否正确，输入 `Y` 并按回车确认。

### 方法二：使用 `useradd` 命令

这是一个底层的非交互式命令。如果不带任何参数直接使用它，创建的用户将没有密码、没有家目录且没有指定 Shell。因此通常需要配合参数使用：

**1. 创建用户并配置基础环境**

```bash
sudo useradd -m -s /bin/bash username
```

- `-m`：强制创建用户的家目录（默认路径为 `/home/username`）。
- `-s /bin/bash`：指定用户的登录 Shell 为 bash。

**2. 为新用户设置密码** 由于 `useradd` 不会自动提示设置密码，你需要手动为其分配：

```bash
sudo passwd username
```

根据提示连续输入两次新密码即可。

---

### 💡 附加操作：授予管理员（sudo）权限

如果你希望新创建的用户能够执行需要管理员权限的命令，可以将其添加到 `sudo` 组中：

```bash
sudo usermod -aG sudo username
```

_(注意：`-aG` 中的 `-a` 代表追加，这能确保用户在加入 sudo 组的同时，不会被移出原有的其他附加组)。_

### 🔍 验证用户是否创建成功

你可以使用以下命令来检查新用户的信息和所属的组：

```bash
id username
```
