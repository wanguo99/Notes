在 Ubuntu 系统中使用 npm 安装 Claude Code，主要分为环境准备、权限配置、正式安装和认证配置四个阶段。为了确保安装过程顺利且避免常见的权限报错，请按照以下步骤操作：

**第一步：安装 Node.js 环境**

Claude Code 依赖 Node.js 运行，要求 Node.js 版本在 18.0 或更高（推荐 v20 或 v22 LTS 版本）。打开终端，依次执行以下命令更新系统并安装 Node.js：

```
# 1. 更新系统包
sudo apt update && sudo apt upgrade -y

# 2. 使用 NodeSource 官方源安装 Node.js v22（LTS长期支持版）
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt install -y nodejs

# 3. 验证安装是否成功
node -v
npm -v
```

如果终端输出了对应的版本号，说明环境准备就绪。

**第二步：配置 npm 全局权限（避坑关键）**

在 Ubuntu 中直接使用 `npm install -g` 安装包时，极易触发 `EACCES: permission denied` 权限报错。**千万不要直接使用** `**sudo**` **强行安装**，否则会导致后续命令无法识别或文件权限混乱。

推荐通过以下命令配置用户级全局目录，一劳永逸地解决权限问题：

```
# 1. 创建用户级全局目录
mkdir -p ~/.npm-global

# 2. 配置 npm 使用该目录
npm config set prefix '~/.npm-global'

# 3. 将全局目录加入环境变量
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc

# 4. 使配置立即生效
source ~/.bashrc
```

**第三步：全局安装 Claude Code**

完成上述权限配置后，即可安全地通过 npm 全局安装 Claude Code：

```
# 全局安装 Claude Code
npm install -g @anthropic-ai/claude-code

# 验证安装结果
claude --version
```

如果终端输出了版本号（如 `2.1.119 (Claude Code)`），即代表安装成功。

_💡 网络优化提示：如果你在国内网络环境下安装速度较慢或出现超时，可以在安装前执行_ `_npm config set registry https://registry.npmmirror.com_` _切换至国内镜像源。_

**第四步：认证与 API 配置**

安装完成后，需要进行身份认证才能正常使用。

1. **常规启动认证**： 在任意项目目录下输入 `claude` 并回车。首次运行会自动弹出一个验证码，并尝试打开浏览器跳转至 Anthropic 官网。登录你的账号（需具备 Claude Pro 订阅或 API 账户）并授权，终端即可自动连接。
    

2. **服务器环境手动配置（无浏览器场景）**： 如果你是在没有图形界面的云服务器上操作，可以通过配置文件直接写入 API Key。
    

- 创建配置目录与文件：
    

```
mkdir -p ~/.claude
vim ~/.claude/settings.json
```

- 在 `settings.json` 中填入以下内容（将 `your-api-key-here` 替换为你实际的 API Key）：
    

```
{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "sk-8098391f61a4d407135e81b5e659d1c105cb4ddb318eb0215ec02a1ef3676503",
    "ANTHROPIC_BASE_URL": "[https://cc-vibe.com](https://cc-vibe.com/)"
  }
}
```

- 保存退出（在 nano 编辑器中按 `Ctrl+O` → `Enter` 保存，再按 `Ctrl+X` 退出）。
    

完成以上步骤后，你就可以在项目目录中直接输入 `claude` 启动工具，开始通过自然语言指令让它协助你编写代码、解释复杂逻辑或处理 Git 工作流了。
