# 1.1.2 配置 ssh

## 1. 安装 openssh
```bash
sudo apt install -y libssl-dev openssh-server openssh-client
```

## 2. ssh 配置文件修改
```bash
sudo cat >> /etc/ssh/sshd_config.d/wanguo.conf << EOF
# 设置密码登录
PasswordAuthentication yes

# 允许root登录
PermitRootLogin yes
EOF
```

## 3. 重启 sshd 服务
```bash
sudo systemctl restart sshd
```
